## Image Assets Generator for Android ##

A tool to generate image assets for android projects.

## Examples & Usage ##
*   Enter the subfolder name at the first prompt (i.e. 'imagesrc').
*   Provide the image width in DP. As a best practice, it should be 1/4 of the image in the folder you provided above.
